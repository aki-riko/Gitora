## 📝 修复方案

### 方案1: 使用AsyncTask封装

```python
# 修改前
success, msg = gitService.commit(message)
if success:
    InfoBar.success(...)

# 修改后
from app.common.async_helper import AsyncTask

def on_success(result):
    success, msg = result
    if success:
        InfoBar.success(...)

AsyncTask.run(
    func=lambda: gitService.commit(message),
    on_success=on_success,
    progress_title='请稍候',
    progress_content='正在提交...',
    parent=self.window()
)
```

### 方案2: 使用SimpleAsyncTask（无进度环）

```python
from app.common.async_helper import SimpleAsyncTask

def on_finished(result):
    success, msg = result
    if success:
        InfoBar.success(...)

SimpleAsyncTask.run(
    lambda: gitService.commit(message),
    on_finished
)
```
