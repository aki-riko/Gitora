### 7. 历史记录缓存（LRU）⭐

**实现位置**: `app/view/history_interface.py` (469-659行)

**新增变量**:
```python
class HistoryInterface:
    CACHE_SIZE = 200  # LRU缓存大小
    
    def __init__(self):
        self._commit_cache = {}      # 缓存字典 {(count, skip): commits}
        self._cache_hits = 0         # 缓存命中次数
        self._cache_misses = 0       # 缓存未命中次数
```

**缓存逻辑**:
```python
def _load_more(self):
    cache_key = (self.PAGE_SIZE, skip)
    if cache_key in self._commit_cache:
        # 缓存命中
        commits = self._commit_cache[cache_key]
        self._cache_hits += 1
    else:
        # 缓存未命中，从Git加载
        commits = gitService.get_log(count, skip)
        # 保存到缓存（FIFO替换策略）
        if len(self._commit_cache) >= self.CACHE_SIZE:
            oldest_key = next(iter(self._commit_cache))
            del self._commit_cache[oldest_key]
        self._commit_cache[cache_key] = commits
        self._cache_misses += 1

def refresh_history(self):
    # 强制刷新时清空缓存
    self._commit_cache.clear()
    self._cache_hits = 0
    self._cache_misses = 0
```

**效果**:
- ✅ 缓存200条提交记录（6-7页）
- ✅ 使用FIFO替换策略（简化的LRU）
- ✅ 统计缓存命中率
- ✅ 强制刷新时清空缓存
- ✅ 减少重复的git log调用
