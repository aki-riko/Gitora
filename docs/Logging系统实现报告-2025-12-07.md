## 🎓 最佳实践

### 1. 日志级别使用
```python
# DEBUG - 详细的执行流程
logger.debug(f"执行Git命令: {cmd}")

# INFO - 关键操作记录
logger.info(f"仓库路径设置成功: {path}")

# WARNING - 警告但不影响运行
logger.warning(f"路径无效: {path}")

# ERROR - 错误但已处理
logger.error(f"Git命令超时: {cmd}")

# EXCEPTION - 异常带堆栈
logger.exception(f"命令执行异常: {e}")
```

### 2. 异常处理规范
```python
# ❌ 错误：静默异常
try:
    do_something()
except:
    pass

# ✅ 正确：记录异常
try:
    do_something()
except Exception as e:
    logger.exception(f"操作失败: {e}")
    raise
```

### 3. 日志消息格式
```python
# ✅ 包含上下文信息
logger.info(f"设置仓库路径: {path}")
logger.error(f"Git命令超时: {cmd}, timeout={timeout}s")

# ❌ 信息不足
logger.info("设置路径")
logger.error("超时")
```
