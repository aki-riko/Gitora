# 远程URL输入界面优化方案

**设计时间**: 2025-12-06 22:14  
**目标**: 让新手更容易配置远程仓库URL  
**方案**: 使用SegmentedWidget + 分段输入

---

## 🎯 问题分析

### 当前问题
用户需要手动输入完整的Git URL，格式复杂：
- HTTPS: `https://github.com/user/repo.git`
- SSH: `git@github.com:user/repo.git`
- SSH with port: `ssh://git@github.com:2222/user/repo.git`

**对新手不友好**：
- 不知道应该用哪种格式
- 容易输入错误（如 `git@host/path` 而不是 `git@host:path`）
- SSH端口配置不直观

---

## 🔧 实现细节

### 1. 使用SegmentedWidget选择协议
```python
from qfluentwidgets import SegmentedWidget

self.protocolSegmented = SegmentedWidget(self)
self.protocolSegmented.addItem("https", "HTTPS", lambda: self._on_protocol_changed("https"))
self.protocolSegmented.addItem("git", "SSH", lambda: self._on_protocol_changed("git"))
self.protocolSegmented.addItem("ssh", "SSH://", lambda: self._on_protocol_changed("ssh"))
self.protocolSegmented.setCurrentItem("https")
```

### 2. 主机名输入
```python
self.hostEdit = LineEdit(self)
self.hostEdit.setPlaceholderText("如: github.com, gitlab.com")
self.hostEdit.textChanged.connect(self._update_url_preview)
```

### 3. 仓库路径输入
```python
self.pathEdit = LineEdit(self)
self.pathEdit.setPlaceholderText("如: username/repository.git")
self.pathEdit.textChanged.connect(self._update_url_preview)
```

### 4. SSH端口（仅SSH协议显示）
```python
self.sshPortEdit = LineEdit(self)
self.sshPortEdit.setText("22")
self.sshPortEdit.setPlaceholderText("默认: 22")
# 初始隐藏，仅SSH协议时显示
```

### 5. URL预览
```python
self.urlPreviewLabel = BodyLabel("", self)
self.urlPreviewLabel.setTextColor(QColor(0, 120, 212), QColor(0, 120, 212))
# 实时显示组合后的完整URL
```

## 💡 用户体验改进

### 优势
1. **可视化选择** - SegmentedWidget直观显示协议类型
2. **分段输入** - 降低输入难度，减少错误
3. **实时预览** - 立即看到完整URL
4. **智能显示** - SSH端口仅在需要时显示
5. **自动组合** - 无需记忆URL格式

### 交互流程
```
1. 用户选择协议 (HTTPS/SSH/SSH://)
   ↓
2. 输入主机名 (github.com)
   ↓
3. 输入仓库路径 (user/repo.git)
   ↓
4. (可选) 修改SSH端口
   ↓
5. 实时看到完整URL预览
   ↓
6. 自动组合并验证
```

class RemoteInterface(QWidget):
    def _setup_ui(self):
        # 1. 协议选择
        self.protocolSegmented = SegmentedWidget()
        
        # 2. 主机名
        self.hostEdit = LineEdit()
        
        # 3. 仓库路径
        self.pathEdit = LineEdit()
        
        # 4. SSH端口（条件显示）

        self.urlPreviewLabel = BodyLabel()

    def _update_url_preview(self):

    def get_remote_info(self):
        # 根据协议类型组合完整URL
        protocol = self.protocolSegmented.currentItem()
        host = self.hostEdit.text()
        path = self.pathEdit.text()
        port = self.sshPortEdit.text()
        
        if protocol == "https":
            url = f"https://{host}/{path}"
        elif protocol == "git":
            if port != "22":
                url = f"ssh://git@{host}:{port}/{path}"
            else:
                url = f"git@{host}:{path}"
        elif protocol == "ssh":
            url = f"ssh://git@{host}:{port}/{path}"
