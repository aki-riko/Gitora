## 🔍 发现的安全问题

### ⚠️ 中等风险（1个）

#### 1. **文件读取未验证路径合法性** (git_service.py 第728-730行)
```python
# 当前代码
full_path = os.path.join(self._repo_path, file_change.path)
with open(full_path, 'r', encoding='utf-8', errors='ignore') as f:
    content = f.read()
```

**风险**: 如果`file_change.path`包含`../`，可能读取仓库外的文件

**建议修复**:
```python
# 验证路径在仓库内
full_path = os.path.join(self._repo_path, file_change.path)
real_path = os.path.realpath(full_path)
repo_real_path = os.path.realpath(self._repo_path)

if not real_path.startswith(repo_real_path):
    raise ValueError("路径不在仓库内")

with open(real_path, 'r', encoding='utf-8', errors='ignore') as f:
    content = f.read()
```

**影响**: 低（Git status返回的路径通常是安全的）
