## 🆕 新增功能详解

### InitRepoGuide 初始化仓库引导窗口 ⭐

**文件路径**: `app/view/init_repo_guide.py`

**核心类**:
```python
class InitRepoGuide(GuideWindow):
    """初始化仓库引导窗口"""
    completed = Signal(str)  # 完成信号
    
    def __init__(self, repo_path: str, parent=None):
        # 添加4个引导页面
        self.addPage(WelcomeInterface())      # 欢迎界面
        self.addPage(UserInfoInterface())     # 用户信息
        self.addPage(RemoteInterface())       # 远程仓库
        self.addPage(FinalInterface())        # 完成界面
```

**引导流程**:
1. **欢迎界面** - 说明引导步骤
2. **用户信息** - 配置用户名和邮箱（可跳过）
3. **远程仓库** - 配置远程仓库URL（可跳过）
4. **完成界面** - 显示配置完成

**特点**:
- ✅ 分步引导，清晰直观
- ✅ 每步可跳过，不强制配置
- ✅ 自动读取全局用户信息
- ✅ 完成后自动保存配置
- ✅ 使用QFluentWidgets Pro的`GuideWindow`组件

```python
def _prompt_add_remote(self):
    """提示添加远程仓库"""
    # 延迟2.5秒显示对话框，等待进度条关闭
    QTimer.singleShot(2500, lambda: self._show_remote_dialog())
```

**作用**: 在初始化仓库成功后调用，延迟显示对话框

#### 方法2: _show_remote_dialog()

```python
def _show_remote_dialog(self):
    """显示远程仓库配置对话框"""
    dialog = RemoteDialog(self.window(), is_new_repo=True)
    if dialog.exec():
        if dialog.validate():
            name, url = dialog.get_remote_info()
            success, msg = gitService.add_remote(name, url)
```

## 🔄 工作流程

### 初始化仓库新流程

```
用户点击“初始化新仓库”
    ↓
选择目录
    ↓
显示进度环“正在初始化Git仓库...”
    ↓
执行 git init（异步）
    ↓
初始化成功
    ↓
进度环变绿“已初始化Git仓库”（2秒后关闭）
    ↓
延迟2.5秒
    ↓
弹出GuideWindow引导窗口 ⭐新增
    ↓
第1步：欢迎界面
    - 说明引导步骤
    - 点击“下一步”
    ↓
第2步：配置用户信息
    - 输入用户名和邮箱（可跳过）
    - 自动读取全局配置
    - 点击“下一步”或“跳过”
    ↓
第3步：添加远程仓库
    - 输入远程名称（默认origin）
    - 输入远程URL（可跳过）
    - 点击“下一步”或“跳过”
    ↓
第4步：完成界面
    - 显示配置完成
    - 点击“开始使用”
    ↓
自动保存配置：
    - 如果填写了用户信息 → git config user.name/email
    - 如果填写了远程仓库 → git remote add
    ↓
显示InfoBar“配置完成”
    ↓
完成
```

## 📝 使用说明

### 初始化新仓库
1. 点击"打开仓库" → "初始化新仓库 (Init)"
2. 选择要初始化的目录
3. 等待初始化完成
4. **自动弹出远程仓库配置对话框** ⭐
5. 填写远程仓库信息或点击"跳过"

### 配置远程仓库
- **远程名称**: 通常使用 `origin`
- **远程URL**: 如 `https://github.com/user/repo.git`
- **跳过**: 稍后可在设置界面添加
