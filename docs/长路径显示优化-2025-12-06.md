## 🎯 问题描述

仓库路径过长时，会挤压右侧的工具栏按钮，导致界面布局混乱。

**问题截图**:
```
DoomsdayHomewardJourney
C:/Users/Kotori/AppData/Roaming/MinecraftPE_Netease/minecraftWorlds/DoomsdayHomewardJourney 分支: master
[打开仓库] [临时保存] [同步] [远程仓库] ← 被挤压
```

---

## 💡 解决方案

### 1. 限制路径标签最大宽度
```python
self.repoPathLabel.setMaximumWidth(500)  # 限制最大宽度为500像素
```

### 2. 启用文本省略
```python
self.repoPathLabel.setSizePolicy(QSizePolicy.Policy.Ignored, QSizePolicy.Policy.Preferred)
```

### 3. 使用省略号显示路径
```python
metrics = QFontMetrics(self.repoPathLabel.font())
elided_path = metrics.elidedText(path, Qt.TextElideMode.ElideMiddle, 480)
self.repoPathLabel.setText(elided_path)
```

### 4. 添加Tooltip显示完整路径
```python
self.repoPathLabel.setToolTip(path)  # 鼠标悬停显示完整路径
```

## ✨ 优化特点

### 1. 智能省略
- 使用 `ElideMiddle` 模式
- 保留路径开头和结尾
- 中间部分用省略号替代
- 例如：`C:/Users/.../Documents/repo`

### 2. 完整信息
- Tooltip显示完整路径
- 鼠标悬停即可查看
- 不影响用户获取完整信息

### 3. 布局稳定
- 路径标签最大宽度500px
- 不会挤压右侧按钮
- 界面布局保持整洁

### 4. 响应式设计
- 使用 `QFontMetrics` 计算实际宽度
- 根据字体大小动态调整
- 适配不同DPI设置

---

## 🔍 技术细节

### QFontMetrics.elidedText()
```python
elidedText(text: str, mode: Qt.TextElideMode, width: int) -> str
```

**参数说明**:
- `text`: 原始文本
- `mode`: 省略模式
  - `ElideLeft`: 左侧省略 `...path/to/file`
  - `ElideMiddle`: 中间省略 `C:/Users/.../file` ⭐推荐
  - `ElideRight`: 右侧省略 `C:/Users/...`
- `width`: 最大像素宽度

### 省略宽度选择
- 标签最大宽度：500px
- 省略计算宽度：480px（留20px边距）
- 确保文本不会超出边界
