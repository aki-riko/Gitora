## 💡 技术实现

### 使用的异步封装

#### 1. AsyncTask（带进度环）
```python
AsyncTask.run(
    func=gitService.xxx,
    on_success=callback,
    progress_title='请稍候',
    progress_content='正在执行...',
    parent=self.window()
)
```
**特点**: 自动显示ProgressInfoBar进度环

#### 2. SimpleAsyncTask（无进度环）
```python
SimpleAsyncTask.run(
    func=gitService.xxx,
    on_finished=callback
)
```
**特点**: 静默执行，适合快速操作
