## 配置迁移

如果用户之前在`AppData/license.key`配置了License，需要手动迁移：

### Windows
```bash
# 创建目录
mkdir %LOCALAPPDATA%\Gitess

# 复制License（如果存在旧文件）
copy AppData\license.key %LOCALAPPDATA%\Gitess\license.key
```

### Linux/macOS
```bash
# 创建目录
mkdir -p ~/.config/Gitess

# 复制License
cp AppData/license.key ~/.config/Gitess/license.key
```
