## 🎯 优化内容

### 修改前（对新手不友好）
```
远程URL: ___________________________________________
         如: https://github.com/user/repo.git
```
- ❌ 需要手动输入完整URL
- ❌ 容易输入错误格式（如 git@host/path）
- ❌ SSH端口配置不直观

### 修改后（对新手友好）⭐
```
协议类型: [ HTTPS ] [ SSH ] [ SSH:// ]  ← SegmentedWidget

主机名: ___________________________________________
        如: github.com

仓库路径: _________________________________________
          如: username/repository.git

SSH端口: __________  ← 仅SSH协议显示
         默认: 22

📋 完整URL预览: git@github.com:user/repo.git
```

- 自动组合完整URL
- 蓝色高亮显示
- 实时更新

### 4. 智能显示/隐藏
- HTTPS协议 → 隐藏SSH端口
- SSH协议 → 显示SSH端口

## 🎨 URL组合逻辑

### HTTPS协议
```python
输入: github.com + user/repo.git
输出: https://github.com/user/repo.git
```

### SSH协议（标准端口22）
```python
输入: github.com + user/repo.git + 22
输出: git@github.com:user/repo.git
```

### SSH协议（非标准端口）
```python
输入: git.example.com + user/repo.git + 2222
输出: ssh://git@git.example.com:2222/user/repo.git
```

### SSH://协议
```python
输入: github.com + user/repo.git + 22
输出: ssh://git@github.com:22/user/repo.git
```
